# Movie Recommender System

Hey! This is my Movie Recommender System project that I built using Python and Machine Learning.

## What does it do?
Simply select any movie you like and it will recommend you 5 similar movies along with their posters. Pretty cool right?

## How I built it
I used the TMDB 5000 Movies dataset and built a content based recommendation system using cosine similarity. The app is built using Streamlit and movie posters are fetched using the TMDB API.

## Files in this project
- `app.py` - the main streamlit app
- `movies.pkl` - processed movies data
- `movies_dict.pkl` - movies dictionary
- `similarity.pkl` - similarity matrix generated from the model

## Technologies I used
- Python
- Pandas
- Scikit-learn
- Streamlit
- TMDB API
